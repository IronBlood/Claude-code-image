# Wiki Overview

- [Goals](./goals.md) describes what the project is for. Mainly maintained by human contributors.
- [Logs](./logs.md) keeps editing logs from any agents. Use the format `[YYYYMMDD HH:mm:ss] MESSAGE_PAYLOAD` to append to the file. Keep the message payload part clear and short.
- [Tasks](./tasks.md) keeps the TODOs to be added to `Dockerfile`. Seeded by human contributors. Each task should solve one single problem.
